Opgave 2

Maak een databank aan met de naam 'pokemon'. 

use pokemon

Opgave 3

Maak in de databank ‘pokemon’ een collection met naam ‘pokedex’. 

db.createCollection("pokedex")

Opgave 4

Voeg een document toe met volgende kenmerken:

Name: Pikachu
Type: electric
Weakness: ground

Laat het object-id door mongodb creëren. 

db.pokedex.insertOne({
  "name": "Pikachu",
  "type": "electric",  
  "weakness": ["ground"]
})

Opgave 5

Voeg een document toe met volgende kenmerken:

Zorg ervoor dat je zelf het object-id toekent. Voor Charmander is dat ‘004’.

Name: Charmander
Type: fire

Probeer eens om volgend document toe te voegen:

Object-id: 004 (als tekst)
Name: Charmander2 
db.pokedex.insertOne({
  "_id": "004",
  "name": "Charmander",
  "type": "fire",
})

Deze functie geeft een error doordat er al een pokemon is met dat id. 

db.pokedex.insertOne({
  "_id": "004",
  "name": "Charmander2"
})

Opgave 6

Voeg een document toe met volgende kenmerken:

Object-id: 005
Name: Charmeleon
Type: fire
Sort: flame
Weakness: ground, rock, water 
db.pokedex.insertOne({
  "_id": "005",
  "name": "Charmeleon",
  "type": "fire",
  "sort": "flame",
  "weakness": ["ground", "rock", "water"]
})

Opgave 7

Voeg via 1 instructie 2 documenten toe:

Object-id: 007
Name: Squirtle
Type: water
Sort: tiny turtle
Evolution: Wartortle, Blastoise


Object-id: 008
Name: Wartortle
Type: water
Weakness: electric 
db.pokedex.insertMany([
  {
    "_id": "007", 
    "name": "Squirtle", 
    "type": "water",
    "sort": "tiny turtle",
    "evolution": ["Wartortle", "Blastoise"]
  },
  {
    "_id": "008", 
    "name": "Wartortle", 
    "type": "water",
    "weakness": ["electric"]
  }
])

Opgave 8

Wijzig het document met id 008.
Voeg hieraan volgend key/value-pair: Evolution: Blastoise 

db.pokedex.update(
  {"_id": "008"},
  {$set: {"evolution": ["Blastoise"]}}
)

Opgave 9

Wijzig het document met id 008.
Voeg een extra weakness toe: grass 

db.pokedex.update(
  {"_id": "008"},
  {$set: {"weakness.1": "grass"}}
)

Opgave 10

Wijzig alle documenten waarin het type ‘water’ is.
Voeg een kenmerk toe: Capacity: torrent 

db.pokedex.update(
  {"type": "water"},
  {$set: {"capacity": "torrent"}},
  {multi:true}
)

Opgave 11

Vervang het document met id ‘004’. Zorg voor volgende inhoud:

Object-id: 004
Name: Charmander
Sort: lizard
Weakness: ground, rock, water 
db.pokedex.update(
  {"_id": "004"},
  {
    "name": "Charmander",
    "sort": "lizard",
    "weakness": ["ground", "rock", "water"]
  },
  {$unset: {"type": ""}}
)

Opgave 13

Wijzig alle documenten waarin het type ‘bug’ is.
Voeg volgende kenmerken toe:

  • Sort: worm
  • Weakness: fire, flying, rock
db.pokedex.update(
  {"type": "bug"},
  {$set:{
    "sort": "worm",
    "weakness": ["fire", "flying", "rock"]
  }},
  {multi:true}
)

Opgave 14

Voeg een document toe met volgende kenmerken:

Object-id: 001  
Name: Bulbasaur  
Type: grass, poison  
Stats: AttackScore: 3
  DefenceScore: 2
  Speed: 3 
db.pokedex.insertOne({
  "_id": "001",
  "name": "Bulbasaur",
  "type": ["grass", "poison"],
  "stats": [{
      "attackScore": 3,
      "defenceScore": 2,
      "speed": 3
  }]
})

Opgave 15

Wijzig bij het document met als Name ‘Metapod’ het kenmerk sort in cocoon. 

db.pokedex.update(
  {"name": "Metapod"},
  {$set: {"sort": "cocoon"}},
  {multi: true}
)

Opgave 16

Voeg bij het document met als Name ‘Metapod’ volgend subdocument toe:

Stats: AttackScore: 1
  DefenceScore: 3
  SpecialAttack: 1 
db.pokedex.update(
  {"name": "Metapod"},
  {$push:
      {"stats":{
          "attackScore":1,
          "defenceScore":3,
          "specialAttack":1
      }}
  }
)
